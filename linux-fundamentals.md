Platform: TryHackMe  
Difficulty: Beginner

## Objective

Learn basic Linux navigation, file handling, and command-line usage for cybersecurity tasks.

---

## Tools Used

• Linux Terminal
• TryHackMe AttackBox

## Commands Practiced

pwd  
ls

cat

---

## Key Concepts Learned

• Navigating directories using CLI
• Viewing and reading files
• Searching for files using find
• Filtering results using grep
• Understanding user permissions basics

---

## Example Task Solution

Example:

Find a specific file:

find / -name filename.txt

Search inside files:

grep "password" filename.txt

---

## Security Relevance

Linux command-line skills are essential for:

• penetration testing
• log investigation
• privilege escalation analysis
• server hardening
• SOC analyst tasks

Understanding filesystem navigation helps locate sensitive files attackers often target.

---

## What I Learned

This room strengthened my confidence working inside Linux environments and improved my ability to locate files quickly during investigations.

These skills are essential for both offensive and defensive cybersecurity roles.
